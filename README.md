# RAG Chatbot Framework

A multi-tenant platform for creating AI-powered chatbots with custom document knowledge bases.

## Project Structure

- `frontend/` - Next.js web application
- `backend/` - FastAPI server
- `chat-widget/` - Embeddable chat widget
- `shared/` - Shared configurations and types
- `docs/` - Documentation

## Tech Stack

- **Frontend**: Next.js + React + TypeScript + Tailwind CSS
- **Backend**: Python + FastAPI
- **Databases**: Supabase (PostgreSQL + Storage), Zilliz (Vector DB)
- **Deployment**: Vercel (Frontend), Render (Backend), Cloudflare Pages (Widget)
